Question-3: What is event delegation in the DOM?

Instead of target multiple child element target it's parent and handle target element event using bubble concept is called event delegation.

Question-4: What is event bubbling in the DOM?

When an event happen in dom it's not get parent element first , first it run it on last child nodes then propagate to their parent nodes like a bubble up . that's why it's called bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element we can document.createElement() to add we can use appendChild() and for removing we have removeElement() option in JavaScript.
